# Получение телефона директора по ИНН

Проект делает только одну задачу:

`ИНН компании -> карточка компании -> клик по кнопке директора -> карточка физлица -> телефон`

## Файлы

- `qr_login.py` - отдельный вход в Telegram через QR для создания сессии
- `get_director_phone.py` - основной рабочий скрипт
- `.env` - настройки Telegram и имя бота
- `results.csv` - файл с результатами, создаётся автоматически

## Установка

```bash
pip install -r requirements.txt
```

## Настройка `.env`

```env
API_ID=123456
API_HASH=your_hash
SESSION_NAME=tg_user
BOT=@your_bot_username
LOG_LEVEL=INFO
RESULTS_CSV=results.csv
```

## Как запускать

### 1. Сначала создать Telegram-сессию

```bash
python qr_login.py
```

Что делает этот скрипт:

- показывает QR-код в терминале
- ты сканируешь его в Telegram
- если включён 2FA, вводишь пароль Telegram
- после успешного входа создаётся файл `tg_user.session`

Если QR-код истёк, скрипт автоматически покажет новый.

### 2. Затем запускать основной скрипт

```bash
python get_director_phone.py
```

После запуска введи ИНН, например:

```text
5421110640
```

Скрипт:

1. отправит `/inn <ИНН>`
2. дождётся карточки компании
3. найдёт кнопку с директором
4. нажмёт её автоматически
5. дождётся карточки физлица
6. вытащит телефон и email
7. сохранит результат в `results.csv`

Выход:

```text
/exit
```

## Если возникает ошибка авторизации

Если `get_director_phone.py` пишет, что сессия не авторизована, нужно снова выполнить:

```bash
python qr_login.py
```

## Что сохраняется в `results.csv`

- ИНН компании
- название компании
- ФИО директора
- ИНН директора
- телефон
- email
- ИНН физлица
